## Installation

MTKPrivacyPods is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MTKPrivacyPods',:git => 'git@192.168.1.58:Joy/iOS-CocoaPods-Privacy.git'
```
